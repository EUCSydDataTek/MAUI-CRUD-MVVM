#	MAUI-CRUD-MVVM med CommunityToolkit.Mvvm ChangeNotification og Commands

En CRUD app, der benytter sig af MVVM design pattern.

Bemærk at der er lavet INPC på Person model klassen, således at UI kan opdateres når der sker ændringer i modellen.
Hvis det fjernes, vil UI ikke opdatere Age når personen gøres ældre.
